A) HTML Structure - top

This theme uses Bootstrap 2.3.2's responsive framework (see full documentation here). The layout for desktop viewports is primarily a two column layout, with a navigation sidebar on the left and the main content on the right. The side bar is in a div with the class "sidebar" and the main content is in it's own div (span9) with no additional classes applied. The main banner at the top of the page (with name and portrait image) are contained within the hero-unit div.

In order to modify the skills ratings section, modify the skill labels, then modify the width to match your skill level. For example:

	
			<div class="row skillRow">
				<div class="skillLabel">Skill Name</div>
				<div class="progress row progressCustom">
					<!-- use bar-success for green, bar-warning for yellow, and bar-danger for red. Then set the width percentage based-->
					<!-- on the skill names, 100% for master, 80% for expert, 60% for skilled, 40% for good, and 20% for fair. Use 0% -->
					<!--for an empty skill bar-->
					<div class="bar bar-success" style="width: 80%;"></div>
				</div>
			</div>

B) CSS Files and Structure - top

The bootstrap CSS has only been slightly modified. In addition to the Bootstrap css, there is also a resume.css file with certain override and extra CSS classes. If you want to add additional classes or modify the current styles, feel free to edit or augment the resume.css file.All of the CSS files can be found in the assets/css/ directory.

C) JavaScript - top

This theme imports two Javascript files.

  1. assets/js/jQuery.js
  2. assets/js/boostrap.js

These files are both required for basic Bootstrap functionality. Neither of the JavaScript files have been modified or augmented in any way. For performance reasons, you can use the minified versions of these files (also included in the project), simply modify the script source tags (lines 299 and 300).
